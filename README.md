# Reto HITSS

Maquetación de una SPA multi-paso en React.

## Tecnologías

- React 19
- Vite
- Tailwind CSS 4
- React Router DOM

## Instalación

```bash
npm install
npm run dev
```

La app corre en `http://localhost:5173`

## Estructura

```
src/
  assets/icons/   → Iconos SVG como componentes
  components/     → Componentes reutilizables
  data/           → Datos estáticos
  pages/          → Vistas (Success)
  styles/         → Estilos globales

  ```
